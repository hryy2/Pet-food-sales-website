# Lovepet - MERN Stack Pet Food Sale Website

## Overview
This project is a fully responsive sale website for selling pet food, developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). 

## Tech Stack

### Frontend
- **React.js**
- **Redux**
- **Tailwind CSS**
- **Axios**

### Backend
- **Node.js & Express.js**
- **MongoDB**
- **JWT (JSON Web Tokens)**
- **Bcrypt.js**


### Backend Setup
Clone the repository:
   cd Pet-Food-Website  
   cd Backend  
   npm install  
   npm run dev  

### Frontend Setup
   cd Pet-Food-Website  
   cd Frontend  
   npm install  
   npm start
